# Portal CN Hub

## 项目简介

Portal CN Hub 是一个用于归档和发布多个独立 HTML 页面的仓库。本仓库不针对任何特定域名或网站，仅作为静态页面的集中存放与展示空间。

这些页面可能包含信息展示、工具界面、文档阅读器、演示页面等多种类型，旨在提供一个简洁、可访问的 HTML 内容集合。

## 目录说明

```
.
├── pages/          # 存放所有独立 HTML 页面文件
├── assets/         # 页面所需的公共资源（如 CSS、JS、图片等）
├── index.html      # 仓库首页，用于导航和页面列表展示
└── README.md       # 本文件
```

- **pages/**：每个子目录或文件对应一个独立页面，命名清晰，便于识别。
- **assets/**：存放多个页面共享的静态资源，避免重复。
- **index.html**：提供简单的页面索引，方便浏览所有归档内容。

## 页面归档说明

本仓库中的 HTML 页面均为独立归档，彼此之间无强依赖关系。每个页面均可直接通过浏览器打开，无需额外服务器环境。

归档原则：
- 页面内容保持原样归档，仅进行必要的兼容性调整。
- 不修改页面核心功能或展示信息。
- 归档时间与原始版本时间可追溯（通过提交记录）。

## 维护说明

- 本仓库由维护者不定期更新，新增或更新页面将提交至 `main` 分支。
- 欢迎通过 Issue 提出页面归档建议或反馈问题，但请注意不要提交任何具体域名、网址或营销内容。
- 不接受直接修改 `pages/` 目录以外核心结构的 Pull Request，除非涉及通用资源或索引页面优化。
- 所有提交应保持页面内容的独立性与完整性。

## 使用方式

你可以通过以下方式访问本仓库中的页面：

1. 直接克隆仓库到本地，在浏览器中打开 `index.html` 或 `pages/` 下的任意 HTML 文件。
2. 通过 GitHub Pages 或其他静态托管服务部署本仓库，即可在线浏览。
